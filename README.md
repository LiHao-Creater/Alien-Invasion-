# Alien Invasion 外星人入侵

外星人要入侵地球了！控制战机发射导弹射击外星人，并躲避外星人的导弹！

## 游戏特性 Game Features

- 🚀 控制战机左右移动和发射导弹
- 👾 成群的外星人会向地球移动并反击
- 🛡️ 护盾系统保护战机免受攻击
- 🎯 得分系统和高分记录
- 🔊 游戏音效（射击、爆炸等）
- 📊 显示生命值、得分和关卡

## 安装 Installation

1. 确保已安装 Python 3.6+ 
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 运行游戏 Run Game

```bash
python alien_invasion.py
```

## 游戏操作 Controls

- **左右箭头键** - 移动战机
- **空格键** - 发射导弹
- **P键** - 开始游戏
- **Q键** - 退出游戏
- **鼠标点击Play按钮** - 开始新游戏

## 游戏规则 Game Rules

- 击毁所有外星人进入下一关
- 避免被外星人或其导弹击中
- 使用护盾保护战机
- 每关难度会逐渐增加
- 游戏结束后可查看最高分

Defend the Earth from alien invasion! 🌍
